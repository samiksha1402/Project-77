Project on Console

Microsoft Visual Studio is an integrated development environment from Microsoft. It is used to develop computer programs, as well as websites, web apps, web services and mobile apps.

Console screen is a screen which is not displayed on the final webpage. To go the console screen, you have to right click on your website and click on inspect. You'll see some written. On the top you'll have some options, by default you will be on the ELEMENTS tab beside that there will be the CONSOLE tab, just click on it. Then you can see the code you have written for the console screen. If you want to learn JavaScript, writing code for the Console is the best way.
